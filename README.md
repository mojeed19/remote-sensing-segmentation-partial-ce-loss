# remote-sensing-segmentation-partial-ce-loss
Weakly‑supervised semantic segmentation for remote sensing images using Partial Cross‑Entropy (pCE) loss. Trains a DeepLabV3+ (ResNet‑50 backbone) on only 1‑10% of pixel‑level labels. Includes experiments on point label density and focal loss gamma. Built with PyTorch, segmentation‑models‑pytorch, and torchgeo.
